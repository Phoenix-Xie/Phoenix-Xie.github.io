在服务器上配置可远程访问的jupyter

安装相关环境

pip3 install jupyter
pip3 install ipython

配置

生成访问密码

在命令行中输入

$ ipython

进入ipython界面

之后输入代码

In[1]: from notebook.auth import passwd
In[2]: passwd() #输入密码
Enter password:
Verify password:
Out[2]: 'sha1:4a2c6bc...'

复制上述sha1加密后的密码’sha1:4a2c6bc…’

在文件夹下新建一个 jupyter_config.py文件，输入如下配置

c.NotebookApp.password = u'sha1:4a2c6bc...'
c.NotebookApp.ip = '*' # 允许任何ip访问，可以设置成特定ip
c.NotebookApp.open_browser = False
c.NotebookApp.port = 8080 # 绑定的端口号，可以任意设置
c.NotebookApp.allow_remote_access = True

打开命令行输入：

jupyter notebook --config=jupter_config.py --allow-root

若要放入后台运行则输入

nohup jupyter notebook --config=jupter_config.py --allow-root & > jupyter_run_statu 2>&1
